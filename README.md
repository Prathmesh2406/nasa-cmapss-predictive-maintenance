# Predictive Maintenance using NASA CMAPSS Dataset
### Remaining Useful Life (RUL) Prediction | Machine Learning | Industry 4.0

This project applies machine learning to predict the Remaining Useful Life (RUL) of turbofan jet engines using the NASA CMAPSS dataset.

## Results (FD001)

| Model | RMSE | NASA Score |
|---|---|---|
| Random Forest | 18.4 | 312 |
| XGBoost | 15.2 | 248 |
| LSTM | 16.8 | 287 |

## Tech Stack
Python, pandas, scikit-learn, XGBoost, TensorFlow/Keras
